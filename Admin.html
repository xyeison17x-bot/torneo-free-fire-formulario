Esta version del proyecto esta pensada para Vercel + Formspree. Las inscripciones se consultan desde el panel de Formspree.

Donde ver inscritos

1. Publica el sitio en Vercel y conecta el formulario con tu endpoint de Formspree.

2. En Formspree entra a tu formulario. Ahi veras todas las inscripciones enviadas.

3. Desde ese panel puedes revisar registros, configurar notificaciones y exportar datos.

Abrir Formspree
Formspree dashboard > Tu formulario > Submissions
